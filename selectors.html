CSS Selector

Basic Selector

Name Syntax Description
Universal Selector * { style properties } Selects all elements. Optionally, it may be restricted to a specific namespace or to all namespaces.
Type Selector element { style properties } matches elements by node name. In other words, it selects all elements of the given type within a document.
Class Selector .class_name { style properties } matches elements based on the contents of their class attribute.
ID Selector #id_value { style properties } matches an element based on the value of the elements id attribute. In order for the element to be selected, its id attribute must match exactly the value given in the selector.

Grouping Selector

Name Syntax Description
Selector list element, element, element { style properties } selects all the matching nodes.
