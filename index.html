Html Table Reference

Tag Table

Tag Name Description
<Table> Table The wrapper element for all HTML tables.
<td> Data Table defines a cell of a table that contains data.
<tr> Table Row defines a row of cells in a table.
<th> Table Header defines a cell as header of a group of table cells.
<theader> Table Head defines a set of rows defining the head of the columns of the table.
<tbody> Table Body encapsulates a set of table rows, indicating that they comprise the body of the table
<tfoot> Table Foot defines a set of rows summarizing the columns of the table.
<Colgroup> Table Column Group defines a group of columns within a table.
<Col> table Column defines a column within a table and is used for defining common semantics on all common cells.

Table Attribute

Attribute Name Description
Colspan Column Span Defines how many columns a td element should span.
Rowspan Row Span Defines how many rows a td element should span.
